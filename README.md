# 11ty-mix

An 11ty starter pack with Tailwindcss and AlpineJS, powered by Laravel-Mix


## Installation

Ensure Node.js v12 is installed, clone the repository and install all modules

```sh
git clone https://github.com/DevForAllFr/11ty-mix
cd 11ty-mix
npm install
```


## Development

```sh
npm run watch
```


## Production

```sh
npm run prod
```
